‚ö†Ô∏è Risk Landscape

Four Critical Risks (Mitigation Required)

R-01: Google Maps "No Competing Service" + Data Caching Lock-in ‚Äî Severity: 5/5 (EXISTENTIAL)

Google Maps Platform ToS ¬ß3.2.3(a) prohibits using Maps/Places content to build a competing mapping product. Place Details, lat/lng, and address data can only be cached for 30 days. All Places API results must display on a Google Map. Google can change terms with 30 days' notice.

Mitigations:

  • Dual-layer architecture: use Google for display; require user-sourced data confirmation to build independent dataset
  • Evaluate Overture Maps Foundation fallback (AWS, Meta, Microsoft-backed; 53M+ places)
  • Legal review of "competing mapping service" definition pre-launch

R-02: Location Data Privacy (GDPR/CCPA/FTC) ‚Äî Severity: 4/5 (HIGH)

FTC enforcement wave against location data brokers (2023-2025). Precise geolocation = sensitive personal info under CCPA. GDPR fines: ‚Ç¨1.64B total in 2023.

Mitigations:

  • Location minimization: never log precise GPS to servers; compute client-side
  • Granular consent architecture pre-launch
  • Data processing agreements (DPA) for restaurant-side B2B ideas

R-03: Cold-Start / Two-Sided Market Death Spiral ‚Äî Severity: 4/5 (HIGH)

App useless until ~150 quality pins per city, 15 active friends on-platform. Consumer social app CAC: $8-25 per install (paid) or $0.50-3 (influencer). 67% of restaurant consumer apps fail within 18 months (CBInsights).

Mitigations:

  • Curated launch: ONE city, 200+ manually curated pins, 3-5 influencer pre-seeds before public launch
  • Invert value prop: make app useful for single user (personal tracker) before social layer
  • For B2B: lead with free analytics, not diner traffic promises

R-04: Google/Incumbent Competitive Response ‚Äî Severity: 3-4/5 (MEDIUM-HIGH)

Google Maps already adding social features (Lists, collaborative maps, AI summaries). Can cross-promote to 1B+ users for free. Copy window: 6-12 months.

Mitigations:

  • Move fast to market (MVP in 6-8 weeks)
  • Build community + trust moat (creator endorsements, local expertise)
  • Focus on "trust network" defensibility, not just features

Implication: All 8 ideas are existentially dependent on Google (R-01). Contingency plan (Overture Maps + OpenStreetMap fallback) must be baked in before Series A.

üõ†Ô∏è MVP Scope & 90-Day Roadmap

Idea #4: Group Dinner Decider (Primary)

MVP (Weeks 1-2): "Swipe Vote"

  • Couples/group members vote on 3-5 restaurant options
  • Shared decision history (past votes, restaurants decided)
  • Supabase backend (reuse V3 stack)
  • iOS + Android via Expo (reuse V2 mobile codebase)
  • Manual restaurant list (top 200 per city)

Phase 2 (Weeks 3-6): "Taste Match"

  • Preference learning: after 10 votes, app learns taste profile
  • Smart recommendations based on past votes
  • Restaurant details (ratings, hours, reservation availability)

Phase 3 (Weeks 7-10): "Restaurant Partnership"

  • Pilot with 10-15 restaurants in launch city
  • Restaurants pay for promoted placement ($50-200/month)
  • OpenTable/Resy affiliate links ($2-5 per reservation)
  • Analytics dashboard for restaurants

Phase 4 (Weeks 11-13): "Group Expansion"

  • Async voting (group members vote over 24-48h)
  • Dietary restrictions + preference tags
  • Friend group expansion (invite friends, set default frequency)

Idea #2: Creator Portfolio (Secondary, Parallel)

MVP (Weeks 1-3): "Manual Concierge"

  • Build 20 creator maps manually for free
  • Measure: % sharing link in bio, clicks per creator per week
  • Validation gate: >15 creators sharing, >100 clicks/week

Phase 2 (Weeks 4-8): "Analytics + Badge" (BUILD ONLY IF VALIDATION HITS)

  • Self-service map creation (import from Google Maps lists, Instagram tags)
  • Analytics dashboard (unique visitors, top restaurants, geographic distribution)
  • "Verified Creator" badge
  • Custom branded portfolio page ([creator-name].mymaps.co)

Phase 3 (Weeks 9-12): "Monetization"

  • Pro tier: $12/month (unlimited, analytics, custom branding, embed widget)
  • OpenTable/Resy affiliate links (10% of booking revenue)
  • "Creator Earnings" feature (estimated annual earnings from referrals)

Phase 4 (Weeks 13+): "Creator Network"

  • Discovery layer: "Find food creators in your city"
  • Social layer: followers, comments, recommendations (long-term)
